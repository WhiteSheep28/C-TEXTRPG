인벤토리 클래스
{
-인벤토리 공간 확보 및 아이템 코드를 저장하는 장소
}

아이템 클래스
{
-인벤토리 배열에 들어있는 숫자에 따라 아이템 코드가 불러와지게 해주는 장소
-아이템 코드는 따로 만들필요 없음 
}

인벤토리 형식
-인벤토리(inventory.cpp)에 들어간다 
-반복문으로 배열을 계속 ++ 해준다 
-반복문을 한번 돌때마다아이템(item.cpp)을 출력한다
-마지막 카운트에 나가기 추가

아이템 입수 형식
-아이템을 얻을 경우 아이템 코드와 갯수를 작성한다
-inventory에 들어와 배열을 한번 돌린다
-inventory안에 들어올 아이템 코드와 같은 숫자가 있다면 그 아이템(item.cpp) 숫자에 갯수를 더한다
-만약에 없다면 한번 더 돌려서 배열의 숫자가 0인 곳에다가 숫자를 넣는다

아이템 사용
-입력받은 숫자 번째의 배열을 들어간다 
-그 배열이 몇번인지 확인 
-번호에 맞는 아이템 사용으로 들어감 (item.cpp)
-아이템 갯수--
-아이템 사용 효과 

아이템 정리
-내 배열의 앞 배열이 0이면 배열을 당김
-0번째 배열의 경우는 제외 1번째 배열부터 당김
